8. 请用一个逻辑表达式来判断某一年是否为闰年。

某一年份如果符合下边 A 或 B 其中一个，我们称之为闰年：
A. 能被 4 整除，但不能被 100 整除；
B. 能被 400 整除。

答：(year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)

解析：|| 左边为 A 条件，右边为 B 条件，符合其中一个，逻辑表达式即为真。
