0. 实现一个洗牌程序。所谓洗牌程序，就是将所有的扑克牌放入一个数组中，要求把数组内的扑克牌顺序随机打乱，然后输出。

答：洗牌的思路其实很简单，就是从所有元素中随机选取一个元素，然后将它与第一个元素进行交换；然后在第二个之后随机选择一个元素与第二个交换，以此类推……直到最后一个元素。这样就能确保每个元素在每个位置的概率都是 1/n 啦~
